# PID-2.5 Scoring Engine

## Objective
Turn feature bundles and audience context into explainable baseline response scores.

## Scope
- direct modeling dimensions for baseline MVP
- confidence estimation
- evidence selection
- score vector generation
- segment-level scoring

## Exit Criteria
- analysis produces machine-readable score vectors with explanations and evidence

## Current Status
Implemented for baseline MVP dimensions. The backend now generates explainable heuristic scores, confidence summaries, evidence spans, and segment-level score vectors.
