# PID-4.2 Preference Capture

## Objective
Capture pairwise or winner-based human preferences between variants and originals.

## Scope
- preference vote contract
- winner and loser linkage
- optional rationale capture

## Exit Criteria
- evaluation can represent preference judgments independently of numeric ratings

## Current Status
Implemented for the current evaluation slice. Evaluation requests now accept structured preference votes with winner/loser linkage and optional reasons.
